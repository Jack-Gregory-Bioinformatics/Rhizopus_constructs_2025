# Rhizopus_constructs_2025
Codon optimisation workflow for fluorescent proteins in **Rhizopus delemar**, based on and adapted from *Emergomyces_constructs_2025* (Edward Wallace, 2025).

This repository contains:

- Scripts to **build a Rhizopus-specific codon-usage table** using RNA-seq data + ribosomal genes  
- DNAchisel workflows to **codon-optimise fluorescent proteins (FPs)**  
- Tools to **align FP protein sequences** and **standardise shared N/C-terminal codon blocks** for primer design  

Adapted and extended by: *Jack Gregory* (2025)

Original work funded by the Wellcome Trust Bioimaging award.

# Full Workflow

## 1. Clone and prepare directory
```
git clone https://github.com/ewallace/Emergomyces_constructs_2025.git
cp -r Emergomyces_constructs_2025 Rhizopus_constructs_2025
cd Rhizopus_constructs_2025
```

---

## 2. Set up `codons2025` environment
```
mamba create -n codons2025 python=3.10 biopython numpy scipy matplotlib pip -c conda-forge -c bioconda
mamba activate codons2025
```

Install DNAchisel stack:
```
pip install dnachisel==3.2.13 dna-features-viewer==3.1.3 geneblocks==1.2.3 flametree==0.1.12 python-codon-tables==0.1.13 python-Levenshtein==0.23.0 rapidfuzz==3.4.0 docopt==0.6.2 tqdm==4.67.1
```

---

## 3. Build genome index & extract CDS
```
mamba create -n histat2_gffread bioconda::hisat2 bioconda::gffread
mamba activate histat2_gffread
```

Build index:
```
hisat2-build alignment/Rdel_genome/Rdel_AB3477.fasta alignment/Rdel_genome/Rdel_hisat2_index
```

Extract all CDS:
```
gffread alignment/Rdel_genome/Rdel_AB3477_EC_removed_variants_removed.gff3   -g alignment/Rdel_genome/Rdel_AB3477.fasta   -x data/Rd_all_CDS.fa
```

---

## 4. Align RNA-seq & generate gene counts
```
chmod +x src/align_and_count.sh
mamba install bioconda::samtools
mamba install bioconda::subread
./src/align_and_count.sh
```

---

## 5. Extract ribosomal genes
```
chmod +x scripts/extract_ribosomal_genes_from_gff3.py
./scripts/extract_ribosomal_genes_from_gff3.py   --gff alignment/Rdel_genome/Rdel_AB3477_EC_removed_variants_removed.gff3   --out data/Rd_ribosomal_gene_ids.txt
```

---

## 6. Build Rhizopus codon-usage table
Install pandas:
```
mamba activate codons2025
mamba install anaconda::pandas
```

Run:
```
./scripts/build_rhizopus_codon_table.py   --counts alignment/results/counts/Rd_featureCounts.txt   --cds-fasta data/Rd_all_CDS.fa   --ribosomal-ids data/Rd_ribosomal_gene_ids.txt   --top-percent 5.0
```

Outputs:
- `data/Rhizopus_delemar_top5pct_plus_ribo_CDS.fa`
- `data/Rd_codon_freqs_by_AA.json`

---

## 7. Codon-optimise FPs using DNAchisel
```
python scripts/codon_adjust_FPs.py
```

Outputs: codon-optimised FP GenBank files in `designs/`.

---

## 8. Protein alignment to find conserved ends
Create multi-FASTA:
```
chmod +x scripts/make_FPs_multifasta.py
./scripts/make_FPs_multifasta.py
```

Align with MAFFT:
```
chmod +x scripts/align_FPs_mafft.sh
./scripts/align_FPs_mafft.sh
```

Shared N-terminal motif discovered:
**MVSKGEE**

---

## 9. Standardise N-terminal (and/or C-terminal) codon blocks
```
chmod 755 scripts/standardise_FP_terminals.py
python scripts/standardise_FP_terminals.py --help
```

Example (standardise N-terminus using mNeonGreen as master):
```
python scripts/standardise_FP_terminals.py   --designs-dir designs   --out-dir designs_Nstd   --mode N   --N-master-gbk designs/mNeonGreenRd_2025-11-26.gbk   --N-motif MVSKGEE
```

Result: all selected FPs now share identical first 21 nt → single forward primer.
